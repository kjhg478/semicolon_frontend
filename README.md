- Role: 팀원
- Position: Front-End
- Works:

1. Authentication

   - 로그인
   - 로그아웃
   - 회원가입

2. Feed

   - 팔로우 한 사람들에게 게시글 노출
   - 날씨, 코로나 API를 이용하여 기능 정보 추가
   - Feed 관리 기능
     - 게시글 관리 (게시글 좋아요, 슬라이드, 게시글 수정 및 삭제 기능)
     - 댓글 기능 (댓글 좋아요, 댓글 작성 및 삭제 기능)
     - 모달 (게시글 자세히 보기)
     - HashTag

3. Profile

   - 팔로우, 언팔로우 기능
   - 프로필 정보 노출
   - 프로필 수정 기능

4. Search
   - 유저 검색
   - 태그 검색
   - 장소 검색
